# Rubio Digital Portfolio 2025

To run locally:
1. `cd ui`
2. `nvm use`
3. `npm run dev`
4. Go to http://localhost:5173/digital-portfolio/

To update:
1. `git status`
2. if needed: `git add file_name`
3. `git commit -am "commit message"`
4. `git push origin main` Or optionally create new branch for a PR
